Resumo
No presente ano letivo foi proposto aos alunos de LI4 que desenvolvessem um projeto com tema a definir pelos mesmos. O tema escolhido deveria respeitar o tema geral que é "Agente de Campo" e possibilitar a construção de um programa de complexidade alta, posteriormente desenvolvido pelos alunos em questão. O nosso grupo optou pelo tema de exploração espacial como tema do projeto a ser desenvolvido. Neste projeto estarão implementados sistemas de leitura de voz, sistema de coordenadas geográficas do astro visistado, entre outras coisas essenciais para o funcionalmento deste projeto, indicados na análise de requisitos. O nosso primeiro passo foi essa mesma análise de requisitos para termos uma ideia da estrutura da relatório e, por consequencia, do projeto. É importante definir a informação presente nos requisitos de uma forma clara, prevenindo futuros enganos e facilitando a discussão dos problemas enfrentados. E ainda vai ser útil na Com isto feito é possível mostrar ao cliente uma ideia de como o projeto se irá organizar e discutir com o mesmo se esta está de acordo com as suas necessidades.

Palavras-Chave: Agente de Campo, Exploração Espacial, Engenharia de Software
